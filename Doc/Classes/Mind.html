Mind

@interface Mind

brief: This is the header file where the super-code is contained.

superclass: NSObject
classdesign No special design is applied here.
helper MindStorage, NeuralMath

The following properties are allocated once during initializtion, in order to prevent frequent memory allocations for temporary variables during the update and backpropagation cycles. Some known properties are computed in advance in order to to avoid casting, integer division and modulus operations inside loops.

There are also methods declared in this file. Refer to their documentations.
