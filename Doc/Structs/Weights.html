Weights

struct Weights {}

@typedef Errors

weights struct help us to create, modify all the weights which are float[] arrays. Accelerate framework doesn’t allow NSMutableArray to other array types, therefore, weights is a nice wraper in memory.

@discussion The values of this structure represent how many sunny, cloudy, rainy, and snowy days existed over the last year. If this was a real app, they could be perfectly used.
